Excel 转换 DBC 的小工具

一个 Excel 转换 DBC 的小工具，方便自己使用，也作为初学前端的练习

多工作表生成

在上传 Excel 文档后，WorkSheet 一栏将识别显示每个工作薄，选择协议填写的工作薄，支持同时选中多个工作薄生成

生成选项

  • 分开生成

Separately 选项在同时选中多个工作薄进行协议生成时，会为每一个工作薄生成单独的 DBC 文件

  • 组合生成

Combined 选项在同时选中多个工作薄进行协议生成时，会将所有工作薄的协议内容整合生成在一个 DBC 文件

信号值表生成

选中此选项后，针对以下类似格式的信号值描述，会为信号生成枚举类型的值表描述

Number DEC
0: Invalid  
1: Enabled  
2: Disabled  
3: Warning
Number HEX
0x0: Invalid  
0x1: Enabled  
0x2: Disabled  
0x3: Warning

提示

相关格式中，枚举值支持十进制和十六进制数字，分隔符支持英文 : 和中文 ： 进行分割
